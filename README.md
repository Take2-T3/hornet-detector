# 🐝 Hornet Detector (スズメバチ検知・監視システム)

カメラ、AI（画像認識）、Discordを連携させ、リアルタイムでスズメバチを検知・通知する監視システムです。Tailscaleを利用した遠隔アクセスにも対応しています。

## 搭載機能

- **自動撮影とストレージ管理**: 0.5秒間隔でカメラ映像を取得し、タイムスタンプを付与して保存。保存上限（1200枚）を超過した場合は古い画像から自動削除します。
- **AIによる物体検知（調整中）**: `best.onnx` モデルを使用し、画像内のハチを検知。確率20%以上で検知とみなし、バウンディングボックス（枠線）と確率を描画します。
- **Discordアラート通知**: ハチを検知した際、枠線付きの画像と警告メッセージをDiscordの指定チャンネルに自動送信します。
- **リモートアクセス**: PythonのWebサーバー機能とTailscaleを組み合わせることで、外出先からスマートフォン等のブラウザ経由で、現地の撮影画像（`captures` フォルダ）を安全に閲覧できます。
